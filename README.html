Pacotes essenciais no R

Obs: Esses foram alguns dos pacotes utilizados no Curso de: Introduo Programao com R para Cincia de Dados



Trabalhamos cada um deles porque representam exatamente o que um Analista, Cientista de dados ou Pesquisador enfrenta no dia a dia.

  • O tidyverse, por exemplo, mostra como possvel manipular, transformar e visualizar dados com fluidez, permitindo que a anlise realmente acontea sem rudo.

  • E, conforme avanamos, entramos em ferramentas que ampliam o alcance do R: consumo de APIs com httr2, criao de servios com plumber, automao com blastula, integrao com Python via reticulate, construo de dashboards com shiny e organizao de pipelines reprodutveis com targets.

  • Para quem vive o contexto de pesquisa, exploramos metan, dbplyr, RMariaDB e outras peas importantes para anlises experimentais e banco de dados.

    Minha inteno foi mostrar como esses pacotes ajudam a resolver problemas reais, facilitando desde o primeiro contato com os dados at a entrega final de resultados.

    No fim, o que quero que fique a viso de que dominar esse conjunto no s saber usar funes, mas conseguir montar um fluxo de trabalho slido, claro e aplicvel em qualquer ambiente de anlise.

timos estudos!

Como reproduzir a estrutura mostrada na imagem

A imagem acima mostra a estrutura de arquivos usada neste repositrio.
Para reproduzir a mesma organizao no seu ambiente, crie uma pasta com a seguinte rvore:

.
 .gitattributes
 pacotes_essenciais.R
 pacotes_essenciais.qmd
 README.md
 README.html
 pacotes.JPG
 temp_hexsession/
     ...

Passo a passo

  1. Crie uma pasta para o projeto no seu computador.
  2. Dentro dela, salve:

  • pacotes_essenciais.R

  • pacotes_essenciais.qmd

  • README.md

  1. Renderize o arquivo Quarto (pacotes_essenciais.qmd) para HTML para gerar o arquivo README.html, se desejar manter a visualizao em HTML.

    • Os pacotes que voc quer visualizar na imagem devem ser carregados com a funo library().

  2. No R, execute:

    library(hexsession)

    hexsession::make_tile()

  • Isso cria a pasta temp_hexsession/, responsvel pela sesso usada na gerao da imagem.

  • Abrindo essa pasta no seu gerenciador de arquivos ou no RStudio, voc ver uma estrutura igual da imagem que acompanha o projeto.
